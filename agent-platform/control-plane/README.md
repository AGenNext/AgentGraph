# Control Plane

This directory holds the platform control layer.

## Core Responsibilities

- identity and session binding
- authority scopes
- policy decisions
- trust state
- intervention gates
- recovery state
- audit and accountability
- one primary agent with many profiles

## Rule

```text
Identity says who.
Authority says what.
Policy says whether now.
Trust says how much autonomy.
Intervention pauses risk.
Recovery restores state.
Audit remembers.
```
