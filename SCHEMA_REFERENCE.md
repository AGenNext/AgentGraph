# AGenNext Schema Reference

This repo uses the unified schema from AGenNext-Enterprise.

## Schema Source
- **Repository**: AGenNext/AGenNext-Enterprise
- **Branch**: schema-org-implementation
- **File**: SCHEMA_ALIGNMENT.md

## Schema Structure

| Role | Description |
|------|-------------|
| **CORE** | Core services, registries, single agents |
| **MULTI_AGENT** | Multi-agent frameworks |
| **PROTOCOLS** | Agent protocols |
| **CODE_REVIEW** | Code review systems |
| **UI** | Chat interfaces |
| **DEPLOY** | Deployment tools |
| **RUNTIME** | Runtime environments |
| **OPS** | Agent operations |
| **SAFE** | Safety systems |

## Usage
All 88 AGenNext repos reference this same schema to ensure ONE DB, ONE SCHEMA, ONE IDENTITY, ONE AUTH, ONE FGA, ONE UI.
