Components

The Content Writing Team consists of modular components for LLM management, agent orchestration, and A2A communication.

Core Modules

core LLM Client core/llm_client.py

Unified client for 40+ LLM providers with OpenAI-compatible API.

ProviderDefault ModelTools
openaigpt-4ocode_interpreter, file_search
anthropicclaude-3.5-sonnetcomputer_use
googlegemini-2.0-flashcode_execution
azuregpt-4enterprise
from core.llm_client import create_llm_client

# Create provider client
client = create_llm_client("openai", model="gpt-4o")

# Generate content
result = client.generate("Write about AI")
print(result.content)

core Registry core/registry.py

Central registry for skills, prompts, tools using DID format.

from core.registry import get_registry

reg = get_registry()

# List available skills
skills = reg.list_skills()
print(skills)

# Get specific skill
skill = reg.get_skill("coding")
print(skill.description)

core Router core/router.py

Routes tasks to appropriate agents based on content type and requirements.

from core.router import ContentRouter

router = ContentRouter()

# Route task to agent
agent = router.route(
    task="Write code",
    content_type="code"
)
print(agent.agent_id)

core Aggregator core/aggregator.py

Aggregates responses from multiple agents.

from core.aggregator import ResponseAggregator

agg = ResponseAggregator()

# Aggregate results
combined = agg.aggregate([result1, result2])
print(combined.content)

core Quality Scorer core/scorer.py

Scores quality of generated content.

from core.scorer import QualityScorer

scorer = QualityScorer()

# Score content
score = scorer.score(content)
print(f"Quality: {score.quality_score}")

core Synthesizer core/synthesizer.py

Synthesizes final output from multiple agent responses.

from core.synthesizer import ResultSynthesizer

synth = ResultSynthesizer()

# Synthesize
final = synth.synthesize(results)
print(final.content)

A2A Protocol

a2a Agent Card a2a/card.py

Agent metadata following A2A protocol specification.

from a2a.card import AgentCard, Skill, Capability

card = AgentCard(
    agentId="openai-agent",
    name="OpenAI Agent",
    description="Coding and reasoning agent",
    url="http://localhost:8000",
    version="1.0.0",
    capabilities=[
        Capability(name="coding", description="Write code"),
        Capability(name="reasoning", description="Logical reasoning")
    ],
    skills=[
        Skill(id="coding", name="Coding", description="Software development")
    ]
)

a2a Messages a2a/messages.py

Message types for A2A communication.

from a2a.messages import (
    Message,
    MessageRole,
    Task,
    TaskStatus,
    TaskState
)

# Create message
msg = Message(
    role=MessageRole.USER,
    parts=[...]
)

# Create task
task = Task(id="task-001", message=msg)

# Check status
print(task.status.state)

a2a Client a2a/client.py

A2A client for agent-to-agent communication.

from a2a import A2AClient

client = A2AClient(agent_url="http://remote-agent:8000")

# Send task
result = client.send_task("Write documentation")
print(result.content)

Orchestrator

orch LangGraph Workflow orchestrator/langgraph_workflow.py

LangGraph-based workflow orchestration.

from orchestrator.langgraph_workflow import create_workflow, run_team_workflow

# Create workflow
workflow = create_workflow()

# Run team workflow
result = run_team_workflow(task="Write content")
print(result)

Workflow Nodes

  • route - Route task to appropriate agent
  • execute - Execute agent task
  • aggregate - Aggregate results
  • score - Quality score results
  • synthesize - Final synthesis
