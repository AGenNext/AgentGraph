Extensibility

Add Custom Agents

Extend the team with your own agents.

class MyAgent(BaseAgent):
    def __init__(self):
        super().__init__(
            agent_id="my-agent",
            name="Custom Agent",
            skills=["skill-id"]
        )

Add Custom LLM Providers

Register new LLM providers easily.

LLM_REGISTRY["my-provider"] = {
    "client": MyProviderClient,
    "default_model": "my-model",
    "api_key_env": "MY_API_KEY"
}

Custom Workflow Nodes

Add custom LangGraph nodes.

@node
def custom_node(state):
    # Process state
    return {"result": process(state)}

Tool Adapters

Wrap external tools as agent tools.

@tool
def my_tool(query: str) -> str:
    return external_api.search(query)
